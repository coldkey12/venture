# aviatickets
Aviatickets desktop application

# Features
- Features online flight tickets purchasing service.
- Interaction with clients through admin/cashier managment panels.
- Clients can buy tickets using passport id.

# Config
- Host: localhost
- Port: 3306
- Driver: MySql

# Instruments
Instruments used: 
- MySql
- JavaFX
- SceneBuilder
- Java 20.0.1
- IntelliJ IDEA

# About managment
Don't be afraid to delete tickets/aircrafts/flights/cities. They don't actually dissapear from the DB, they just switch their status to either INACTIVE/ACTIVE/DEAD. If killed accidentaly, you can change status manually through the db.
